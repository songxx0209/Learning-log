# 操作符、变量、作用域



## 操作符

```
'1' - -'1'; // 2

从左往右执行：
'1' - (-1);
// 减回隐式转换 1 - -1 = 2
```



```
'1' - -'1'+'1'; // ‘21’

从左往右执行：
2 + ‘1’；
// 加会把数字转换为字符串处理
```



## 变量

js中变量分为基本类型、引用类型；

```
var a = 1;
var b = a;
b = 2;
console.log(a, b); // 1, 2
```



```
var a = { n: 1 };
var b = a;
a.x = 2;
b.x = '3';
console.log(a, b); // { n: 1, x: '3' }, { n: 1, x: '3' }

// a,b操作的都是同一个对象
```



ECMAScript中所有函数的参数都是按值传递；

```
function addTen(num) {
    num += 10;
    return num;
}
var count = 20;
var result = addTen(count);
alter(count); // 20
alert(result); // 30
```



## 作用域

javascript采用的是词法作用域 （静态作用域）；

**词法作用域：**函数的作用域在函数定义的时候就决定了

**动态作用域：**函数的作用域是在函数调用的时候才决定的

```
var value = 1;

function foo() {
    console.log(value);
}

function bar() {
    var value = 2;
    foo();
}

bar();

// 1
```
