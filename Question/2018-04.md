**linux scp命令：**

```
scp -r ./1 cdweb@114.141.173.17:/home/cdweb/ss
// 第一次执行
// 将文件夹1中的文件拷贝到ss文件夹中；

scp -r ./1 cdweb@114.141.173.17:/home/cdweb/ss
// 再次执行，并没有覆盖之前位置的文件
// 而是在ss文件夹中新增了一个1文件夹
```

```
scp -r ./1 cdweb@114.141.173.17:/home/cdweb/ss/

scp -r ./1/ cdweb@114.141.173.17:/home/cdweb/ss

scp -r ./1/ cdweb@114.141.173.17:/home/cdweb/ss/
// 与上面命令相同的结果
```

**原因：是应为开始时远端服务器没有ss这个文件夹，第二次上传才会正确复制文件夹到远端文件夹；**
